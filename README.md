# 🖥️ Status Bar Setup for Ubuntu

One script to install and configure [indicator-sysmonitor](https://github.com/fossfreedom/indicator-sysmonitor).

```
🚀 Net | 💻 CPU 🌡️ Temp | 🎮 GPU VRAM 🌡️ Temp | 🧠 Mem 🔁 Swap | 💾 Disk
```

## Usage

```bash
git clone <your-repo-url> && cd Status-Bar && bash setup.sh
```

## Requirements

- Ubuntu 24.04+
- NVIDIA GPU with `nvidia-smi`
